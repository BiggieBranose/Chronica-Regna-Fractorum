# Vulkan

"why vulkan?"
Well, we want to really understand how the whole rendering pipeline works and keep full control over each part of it, so that we can optimize things better. We decided not to use OpenGL because the performance wasnt good enough for what we wanted, even though it’s easier to work with. we also decided not to use DirectX, since we want the project to stay truly cross‑platform and not tied to one system.
